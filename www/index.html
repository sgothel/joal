Overview

The JOAL Project hosts a reference implementation of the Java bindings for OpenAL API, and is designed to provide hardware-supported 3D spatialized audio for games written in Java.

This project also hosts the Sound3D Toolkit, a high level API for spatialized audio built on top of the OpenAL bindings. This toolkit is designed to provide access to all the features of OpenAL through an intuitive, easy to use, object-oriented interface.

OpenAL-Soft

JOAL is tested using OpenAL-Soft the cross-platform, software implementation of the OpenAL 3D audio API.
